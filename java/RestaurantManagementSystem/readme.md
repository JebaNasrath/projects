
# Restaurant Management System (Java)

A simple Java console-based application to manage basic restaurant operations like viewing menu items, taking orders, generating bills, and maintaining order details.  
This project was built to practice core Java and Object-Oriented Programming (OOP) concepts.

## Features
- Display menu items with prices
- Take customer orders (select item + quantity)
- Generate total bill amount
- Basic order summary / receipt
- Input validation for menu choice and quantity (basic)

## Tech Stack
- Java (Core Java)
- OOP Concepts (Classes, Objects, Encapsulation)
