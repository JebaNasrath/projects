# Student Request Tracking System

A simple web application built using HTML, CSS, and JavaScript that allows
students to raise service requests and track their status.
This project was developed to practice frontend fundamentals and client-side logic.

## Project Overview
The system enables students to:
- Submit a request with name and issue description
- Assign and update request status (Open, In Progress, Closed)
- View all submitted requests dynamically

This project simulates a basic IT support / service request workflow.

## Features
- Form-based request submission
- Client-side validation for required fields
- Request status flow: Open → In Progress → Closed
- Dynamic UI updates using JavaScript
- Add and delete requests

## Technologies Used
- HTML (structure and forms)
- CSS (styling and layout)
- JavaScript (validation, DOM manipulation, status updates)
